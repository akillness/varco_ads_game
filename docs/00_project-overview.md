# VARCO Agent SAGA Promo - 프로젝트 개요

## 목표
- `VARCO3D_UGC_생태계게임_v4.pdf`의 핵심 제안(UGC 기반 Agent Sandbox)을 웹 홍보 게임으로 구현
- VARCO API 서버와 연결 가능한 구조로 제작
- 2026년 내 저비용/고효율 홍보형 게임 PoC를 즉시 시연 가능한 수준으로 제공

## 핵심 가치
1. Make-to-Play: 유저 생성 에이전트 즉시 플레이
2. API 기반 콘텐츠 확장: 사운드/3D 생성
3. 커뮤니티 확장: 로그/마켓플레이스/B2B 모델

## 산출물
- 플레이 가능한 웹 게임(React + Vite)
- VARCO API 프록시 서버(Express)
- 역할별 합의 문서, API 사용 가이드, 레퍼런스 분석 문서
